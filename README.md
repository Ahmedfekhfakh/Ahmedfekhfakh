## 🚀 Featured Projects

### 📊 [GamePulse](https://github.com/Ahmedfekhfakh/stock-game-correlation) — Big Data Analytics Pipeline
> Game popularity × stock price correlation · Télécom Paris, 2026  
> `PySpark` · `XGBoost` · `Airflow` · `Elasticsearch` · `Kibana` · `LocalStack S3` · `Docker`

Fully containerized 8-service Data Lake pipeline ingesting **Steam, Twitch & Yahoo Finance** data. Spark computes 7-day rolling Pearson correlations and trains an XGBoost classifier across 5 signal classes (STRONG_POSITIVE → STRONG_NEGATIVE). Achieved **79% weighted F1** on 647 rows / 17 tickers with a 5-minute live Kibana dashboard refresh — zero cloud dependency via LocalStack S3.

---

### 🏗️ [MapReduce Distributed System](https://github.com/Ahmedfekhfakh/Mapreduce-distributed-system) — Amdahl's Law Validation
> Custom MapReduce from scratch · 1.3 TB CommonCrawl · Télécom Paris, 2024  
> `Python` · `TCP Sockets` · `Linux VMs` · `Distributed Computing`

Built a Master-Worker MapReduce engine from scratch (no Hadoop) deployed across 3 Linux VMs via SSH. Experimentally validated Amdahl's Law: measured **21% serial fraction**, achieving **2.31× speedup** (77% efficiency) with 3 workers. Identified SHUFFLE as the primary bottleneck — scaling to only 1.94× — and proved shuffle time depends on **key cardinality, not data volume**.

---

### 🔐 [FraudGuard](https://github.com/Ahmedfekhfakh/Fraudguard) — MLOps Fraud Detection Pipeline
> Production MLOps stack · Télécom Paris DATA713, 2025  
> `LightGBM` · `Isolation Forest` · `FastAPI` · `MLflow` · `Airflow` · `Docker` · `Kubernetes`

End-to-end MLOps pipeline for real-time credit card fraud detection with automated retraining, drift detection, and a Streamlit monitoring dashboard. Fully migrated from Docker Compose to Kubernetes.
